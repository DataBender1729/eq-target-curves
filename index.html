EQ Target Curve Generator






Instructions:
Pick four Hz/dB reference points and 3 functions to blend between them.
Click generate to create a target curve file to download and use in your room correction software (e.g., Dirac Live).
If you want to apply PEQ filters to the target curve, copy the output and take it to the Target Curve PEQ tool

Curve types:
  • Linear: Just a line from the starting dB to the end dB
  • Curved: Very similar to the popular harman curves. Amplifies the low end of the curve and then drops quickly.
  • Cosine: Drops to the halfway dB point after half the Hz-distance in a (co)-sine shape
